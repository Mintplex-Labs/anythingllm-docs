# AI Agents

Agents are LLMs that have the ability to use tools to perform tasks to complete your requests. There are a number of pre-provided tools by default, but you can also create your [own custom tools](/agent/custom/introduction), [use MCPs](/mcp-compatibility/overview), or even use [Agent Flows](/agent-flows/overview) to build your own custom tools.

AnythingLLM will automatically determine if your LLM can use tools so that every chat is more intelligent. However, now all LLMs can use tools, you can still choose to use the `@agent` directive at the start of your chat to start an agentic chat session.
