## New Features:

- [x] Custom Slash Commands
- [x] UI updates
- [x] [LiteLLM](https://github.com/BerriAI/litellm) Support for LLM
- [x] Widescreen chat window support
- [x] TTS support for response (native, OpenAI, and 11Labs)
- [x] Chat response regeneration
- [x] Website (depth) scraper tool
- [x] API Key support for TextWebGenUI

## Fixes:

- Confluence workspace support
- Workspace not showing updates on changes save

## What's Next:

- LiteLLM Embedder model support
- VoyageAI Embedder model support
- RAG response improvement/overhaul
- Expand Search API for `@agent`
- Message editing (prompt **and** response)
- Document picker search
