## New Features:

- [x] [New ability to easily create fine-tuned models!](/fine-tuning/overview)
- [x] Ability to locally import `GGUF` files into the native LLM (works with any GGUF)
- [x] Added GitLab repo loading support
- [x] Added AWS Bedrock LLM support

## Fixes & Improvements:

- Bumped known models for Perplexity
- Dependency pruning and security patches
- GroqAI now gets available models from their hosted endpoint (finally)
- Added support for the `keep_alive` parameter for external Ollama.
