## Notable Improvements

### Automatic Mode for native tool calling

For [Select providers](/features/chat-modes#available-chat-modes) that support native tool calling, you no longer need to use `@agent` to use tools. You can now just use the tools directly in your prompt.

If your prompt input does not have the "@" symbol, your chats will automatically use tools as needed.

## Intelligent Tool Selection

We have added a new feature called [Intelligent Tool Selection](/agent/intelligent-tool-selection). This feature allows you to load **unlimited** tools for your agent to use into context with better performance and save up to 80% on token usage every single chat.

We have added a new built-in agent for [Document Generation](/agent/usage#document-generation-agent). With document generation, you can generate text files, PDFs, Excel files, Docx, and even entire PowerPoint presentations.

## Telegram Bot

AnythingLLM Docker and Desktop now support a [Telegram bot](/channels/telegram) so you can connect to your AnythingLLM instance anywhere in the world.

**Supports**:
- Text chat (streaming & thinking)
- Image understanding
- Voice messages & Attachments
- Automatic mode and @agent support
- Workspace and thread selection
- Model selection
- Citations
- Any agent skill available in AnythingLLM
