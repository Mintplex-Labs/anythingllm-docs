## Improvements

- [x] Groq Speech-to-Text support (whisper models)
- [x] Brave Search web-search provider
- [x] FastCRW web-browsing support
- [x] Workspace settings spacing and design layout changes
- [x] Improved UI/UX for mass delete of threads - now only shows when hovering over the thread container instead of everytime you press CMD/CTRL
- [x] Migrated AWS Bedrock provider to OpenAI SDK for compatibility and removed it's packages.

## Bug Fixes

- [x] User prompt edit not appearing in agent mode after message completion
- [x] Omit temp in Claude for Anthropic for Claude 4.8
- [x] Fix Ollama keepalive with auth header when using API key
- [x] Fix logic in Embed allowlist (self-hosted only)
- [x] Fix bug where file attachments were not being displayed in chat when used during agent chat as citations
